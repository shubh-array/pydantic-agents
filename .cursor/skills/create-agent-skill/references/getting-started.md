# Getting started

1. Set the active adapter in `config/active_agent` (`cursor` or `claude_code`) or override with env `AGENT_SKILL_ADAPTER`.
2. Read `references/execution-contract.md` for the exact command for each phase (A–F).
3. For Cursor-specific CLI patterns, open `adapters/cursor/recipes.md`. For Claude Code, open `adapters/claude_code/recipes.md`.
